Tela de Criativos

Tela para realizar o Upload de criativos de forma manual e armazená-los no Google Cloud Storage.

Tela de Criativos

  • Desafio: Criar uma tela para que seja possível realizar o Upload de criativos de forma manual.
    Os criativos podem ser imagens, vídeos ou links de postagens. A taxonomia e o trackcode das campanhas são digitados na tela. O processo também transforma vídeos em gifs.

    Os arquivos de imagens e gifs são armazenados no Storage. Os dados de texto (taxonomia e trackcode) são armazenados no BigQuery.

  • Resultados:Agilizar o processo manual do Upload do criativo que não foi possível extrair de forma automática. Também preserva o histórico do arquivo, tornando-o acessível para uso em dashboards e relatórios.

  • Meu papel: Desenvolvimento do script de automação.
  • Clientes: Samsung
  • Tecnologias: Python | BigQuery | Google Storage | API | C# | Angular
  • Data: Janeiro de 2025

Criativos - Meta

Automação para extrair e armazenar os criativos de Meta em um Storage do Google Cloud.

Criativos - Meta

  • Desafio: Realizar o download dos criativos das campanhas nos formatos (jpeg, png, vídeo) e armazenar permanentemente. O processo também transforma vídeos em gifs e realiza o Web Scraping para extrair a taxonomia e o trackcode das campanhas.

    A automação coleta esses arquivos da biblioteca de criativos da META e os armazena no Storage. Os dados de texto (taxonomia e trackcode) são armazenados no BigQuery.

  • Resultados: Preservação do histórico de arquivos, evitando que expirem e tornando-os acessíveis para uso em dashboards e relatórios.

  • Meu papel: Desenvolvimento do script de automação.
  • Clientes: Samsung
  • Tecnologias: Python | BigQuery | Google Storage | API | C# | Web Scraping
  • Data: Julho de 2024
